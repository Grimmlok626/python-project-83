# Проект Python Project 83
Этот проект — веб-приложение для анализа страниц на SEO с помощью Flask и базы данных PostgreSQL.

## Установка
1. Клонируйте репозиторий:
git clone https://github.com/Grimmlok626/python-project-83.git
cd python-project-83
2. Создайте виртуальное окружение:
python3 -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows
3. Установите зависимости:
pip install -r requirements.txt

## Запуск проекта
Запустите Docker контейнеры:
docker-compose up -d
